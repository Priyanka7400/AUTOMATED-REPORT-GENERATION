# AUTOMATED-REPORT-GENERATION
## Project Title
PDF Report Generator using Python
Description
This project reads data from a CSV file, analyzes it, and generates a formatted PDF report using Python.
The PDF is created using the ReportLab library.
## Technologies Used
Python 3
ReportLab
CSV file handling
## Files in the Project
pdf_repot.py → Python script to generate PDF
sample_data.csv → Input data file
student_report.pdf → Generated output PDF
